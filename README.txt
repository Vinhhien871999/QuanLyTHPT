Hướng dẫn cài đặt và sử dụng 

Bước 1: Tải về từ Github https://github.com/GroupMTA2020/SanKham.git
Bước 2: Mở Folder SanKham từ thư mục tải về
và vào thư mục bin : ..\SanKham\THPT\THPT\bin
Bước 3: Mở Debug folder và chạy file THPT.exe để sử dụng ứng dụng
Bước 4: Nhập user và password để đăng nhập vào và sử dụng.
